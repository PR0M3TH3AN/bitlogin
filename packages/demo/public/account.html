Try a real account

This page embeds the exact <bitlogin-auth> element any static site would use — nothing special is happening behind the scenes.

Create an account to see a generated passphrase and a 12-word recovery phrase, sign a test event, rotate your password (old capsule is tombstoned), and export a recovery file — or reload this page and sign back in from a "clean" browser state.

Uses BitLogin's built-in public relay list by default. Point vault-relays / discovery-relays attributes at your own relays for production use.

BitLogin is an experimental alpha protocol. Accounts created here live on public Nostr relays — don't use a real password or phrase you rely on elsewhere.
